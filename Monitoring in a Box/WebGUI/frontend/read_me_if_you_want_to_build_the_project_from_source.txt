1. Install nodejs and npm on your machine
2. Run this command to create a reactjs project skeleton "npx create-react-app <your project folder name>"
3. Change directory into your new project folder and overwrite files with the files in this repository (e.g. do not remove the node_modules folder)
4. Run all commands listed in the 'to do.txt' file
5. Build the project "npm run build"
6. Run the build file "serve -s build"
